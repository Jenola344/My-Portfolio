# **App Name**: Web3 DevRel Catalyst

## Core Features:

- Animated Hero Section: Interactive intro with motion typography showcasing 'Building the bridge between Developers and Web3 Ecosystems' and a cursor-reactive 3D animation.
- Storytelling About Section: Parallax storytelling section detailing the journey from developer to DevRel, enhanced with smooth transitions.
- Expertise Showcase: Split layout section to present Frontend, Smart Contract, and DevRel expertise with animated skill cards that respond to scroll.
- Interactive Web3 Project Demos: Highlights interactive Web3 projects with live demo links and code previews. Incorporate a basic smart contract interaction demo (e.g., minting a token or verifying wallet ownership) using WalletConnect with Celo, Ethereum, and Polygon.
- Community Impact Display: Showcase public talks, workshops, and hackathons with embedded videos and animated badges. Include data pulled directly from the blockchain using a tool.
- Token-Gated Contact Form: An interactive contact form that mimics minting a token on submission, plus a wallet connect feature and animated CTA button.
- Dynamic Theme Switching: Option to toggle between dark and light themes, complete with animated route transitions for a seamless experience.

## Style Guidelines:

- Primary color: Holographic Electric Indigo (#6F00FF) for a modern and technologically advanced feel.
- Background color: Dark charcoal grey (#121212) providing a stark contrast to make the neon elements pop.
- Accent color: Cyberpunk Fuchsia (#FF00BF) for interactive elements and highlights, complementing the indigo.
- Body and headline font: 'Space Grotesk' sans-serif for a tech-inspired, elegant feel.
- Use glowing, neon-style icons that resonate with Web3 and developer themes.
- Dynamic layouts with sections that flow via horizontal scroll or parallax effects for storytelling.
- Fluid, immersive animations via Framer Motion, GSAP, and ScrollTrigger, telling a cohesive story with each transition.
